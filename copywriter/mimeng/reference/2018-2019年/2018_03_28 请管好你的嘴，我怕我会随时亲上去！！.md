# 请管好你的嘴，我怕我会随时亲上去！！

**日期**: 2018-03-28

---

因为即将有一大波撩人的情话朝你袭来。等你看过别的男生说的情话之后，就会觉得你的男友可有可无了。哎，连我这个老阿姨都想重新恋爱一次了好吗。——迷你蛋——小黎我们在准备婚礼事宜的时候他问我要生辰八字去选日子，我们之间的某句对话。——华哥的小迷妹——赵小糖我23岁，很喜欢一个人，真的真的很喜欢很喜欢。有天我小心翼翼地问他：“如果我追你，能追上吗”发出去之后就后悔了，因为他平时太高冷了……我连忙开玩笑地补了句“你体育那么好，我肯定追不上的……哈哈哈”他回：“我不跑，傻站着”——小颜——王小喵朋友和男友吵架了，她决定分手。那时男方出资买了房，两人本来打算结婚的。男友说，分手后要把房子过户到朋友这里。朋友问，为什么？男友回答，因为以后断了联系，至少我还知道你住哪里啊！——焦糖玛奇朵我和男朋友异地恋，广州和北京，今年元宵节在一起的，希望我们能一直走下去。最近看才华有限青年的歌单，喜欢上了丢火车乐队的《晚安》，里面有一句歌词：你会不会突然地想起我。我有一天发了这句歌词给他。他说，不会那么突然，因为时时挂念。——赵雅思——小杨——大琳琳男：我问你一个问题。男：爱一个人是什么样的？男：你知道吗？女：我知道啊，就是会发藏头对话给她。——假麦——娃哈哈实不相瞒，我在看征集的时候也收到了很多同学发给我的情话，其中一条是这样的：简直就是暖！炸！了！
